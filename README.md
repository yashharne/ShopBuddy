# ShopBuddy

ShopBuddy is a modern e-commerce web application built to provide a seamless shopping experience for users. With a robust tech stack and user-friendly interface, it showcases the technical intricacies involved in enhancing online shopping experiences.

## Tech Stack

- **Next.js**: A React framework for building efficient and scalable web applications.
- **Tailwind CSS**: A utility-first CSS framework for building responsive and customizable user interfaces.
- **DaisyUI**: A collection of components for Tailwind CSS, enhancing UI development.
- **MongoDB Atlas**: A cloud-based database service for MongoDB, ensuring scalability and flexibility.
- **Prisma**: A modern database toolkit for TypeScript and Node.js.
- **TypeScript**: A typed superset of JavaScript that adds static types to the language, improving code quality and developer productivity.
- **NextAuth**: Authentication library for Next.js applications, providing a secure and extensible authentication solution.
- **Zod**: A TypeScript-first schema declaration and validation library for building robust data models.

## Features

- **User Authentication**: Secure user authentication powered by NextAuth for a seamless login experience.
- **Product Catalog**: Browse through a diverse catalog of products with detailed information.
- **Shopping Cart**: Add products to the cart, manage quantities, and proceed to a hassle-free checkout.
- **Responsive Design**: Enjoy a consistent shopping experience across various devices with a responsive design.

## Getting Started

To run ShopBuddy locally, follow these steps:

1. Clone the repository:

```bash
git clone https://github.com/yashharne/ShopBuddy.git
```

2. Navigate to the project directory:
```bash
cd ShopBuddy
```

3. Install dependencies:
```bash
npm install
```

4. Set up environment variables:
Create a `.env.local` file in the root directory and add the necessary environment variables.


5. Run the development server:
```bash
npm run dev
```

6. Visit http://localhost:3000 to view ShopBuddy in your browser.
